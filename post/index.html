编码规范

July 23, 2019 in 规范
风格 使用 arrow 函数代替匿名函数表达式。 只要需要的时候才把 arrow 函数的参数括起来。 比如，(x) => x + x 是错误的，下面是正确的做法： i. x => x + x ii. (x,y) => x + y iii. <T>(x: T, y: T) => x === y 总是使用 {} 把循环体和条件语句括起来。 开始的 { 总是在同一行。 小括号里开始不要有空白. 逗号，冒号，分号后要有一个空格。比如： i. for (var i = 0, n = str.length; i < 10; i++) { } ii. if (x < 10) { } iii. function f(x: number, y: string): void { } 每个变量声明语句只声明一个变量 （比如 使用 var x = 1; var y = 2; 而不是 var x = 1, y = 2;）。 else 要在结束的 } 后另起一行。
